# B1 Auth User Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first authentication and user-service backend slice with Flyway migrations, separated persistence entities, DTOs, and minimal register/login/current-user APIs.

**Architecture:** Shared fixed table fields live in `foodmap-common` as `BaseEntity`. Service database rows are represented by `infrastructure.persistence.entity` classes and never exposed from controllers. Controllers use DTOs and delegate orchestration to application services.

**Tech Stack:** Java 21, Spring Boot, Maven, Bean Validation, PostgreSQL Flyway SQL.
